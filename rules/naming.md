# Naming Rules

Use the Forge thematic naming mappings:
- Forge = central runtime
- Arsenal = shared library system
- Blueprint = goal or plan
- Workpiece = unit of work
- Artisan = AI worker (PlannerArtisan, BuilderArtisan, ReviewerArtisan, etc.)
- Guild = collection of artisans
- Workshop = execution context
- Anvil = state machine
- Bellows = scheduler/heartbeat
- Archive = long-term memory
- Codex = project knowledge
- Instrument = tool
- Gateway = external connector
- Inspector = evaluator
- Assayer = verifier
- Artifact = final output
- Chronicle = decision log
- HammerStrike = domain event
- FoundryMetrics = metrics
